**Goal:** Add parser unit tests (UCI protocol compliance) and real-engine
integration tests to `@echecs/uci`.

**Architecture:** Two new test files alongside the existing
`src/__tests__/index.spec.ts`. Parser tests are pure synchronous assertions
against raw UCI strings. Integration tests create a real `UCI` instance via
`UCI_ENGINE_PATH` env var and assert on emitted events; the suite is skipped
when the env var is absent.

**Tech Stack:** Vitest 4, TypeScript 5, `@vitest/eslint-plugin`, Node.js
child_process (via existing `Process` class).

**Step 3a (if failures): Fix the `option` parser**

The `option` parser uses
`extract(['default', 'max', 'min', 'name', 'type', 'var'])`. The problem is that
`extract` splits on spaces and treats each UCI keyword as a key boundary — so a
multi-word option name like `Clear Hash` gets split at `Hash` if `Hash` is not a
known key. Since `Hash` is not in the keyword list, it actually accumulates into
`name`'s value — **verify this first** before changing anything. Run the test
and read the actual output.

If the name does not include the full multi-word value, rewrite
`src/parser/index.ts` to use a dedicated `option` parser instead of `extract`:

```typescript
// src/parser/option.ts
function option(value: string): Partial<Record<string, string>> {
  const tokens = value.split(' ');
  const keys = new Set(['type', 'default', 'min', 'max', 'var']);
  const result: Partial<Record<string, string>> = {};

  let key = 'name';
  for (const token of tokens) {
    if (keys.has(token)) {
      key = token;
    } else {
      result[key] =
        result[key] === undefined ? token : `${result[key]} ${token}`;
    }
  }

  return result;
}

export default option;
```

Update `src/parser/index.ts` to import and re-export `option` from `./option.js`
instead of using `extract`.

## Task 6: Integration tests

**Files:**

- Create: `src/__tests__/integration.spec.ts`

**Step 1: Write the integration test file**

```typescript
import { afterEach, beforeEach, describe, expect, it } from 'vitest';

import UCI from '../index.js';

const ENGINE_PATH = process.env['UCI_ENGINE_PATH'];

describe.skipIf(!ENGINE_PATH)('UCI integration', () => {
  let uci: UCI;

  beforeEach(() => {
    uci = new UCI(ENGINE_PATH!);
  });

  afterEach(() => {
    uci.stop();
  });

  it('emits uciok on init', () =>
    new Promise<void>((resolve) => {
      uci.on('uciok', () => resolve());
    }));

  it('emits id with name and author', () =>
    new Promise<void>((resolve) => {
      uci.on('id', (data) => {
        if (data.name || data.author) resolve();
      });
    }));

  it('emits at least one option', () =>
    new Promise<void>((resolve) => {
      uci.on('option', () => resolve());
    }));

  it('emits info events after start()', async () => {
    await new Promise<void>((resolve) => {
      uci.on('uciok', () => resolve());
    });

    return new Promise<void>((resolve) => {
      uci.on('info', () => resolve());
      uci.start();
    });
  });

  it('emits bestmove after a move', async () => {
    await new Promise<void>((resolve) => {
      uci.on('uciok', () => resolve());
    });

    await uci.start();

    return new Promise<void>((resolve) => {
      uci.on('bestmove', ({ move }) => {
        expect(move).toBeTruthy();
        resolve();
      });
      uci.move('e2e4');
    });
  });

  it('does not emit error after stop()', async () => {
    await new Promise<void>((resolve) => {
      uci.on('uciok', () => resolve());
    });

    const errors: Error[] = [];
    uci.on('error', (e) => errors.push(e));

    await uci.stop();

    expect(errors).toHaveLength(0);
  });
});
```
